# LinkedIn Scroll Fix

An unofficial browser extension that fixes slow scrolling on [LinkedIn.com].

The issue seems to occur because LinkedIn sets a very small font size, which can make page scrolling
much slower in some browsers. This extension fixes that by forcing a normal font size in the main
content area.
